# RepoRevive

AI-powered tool that clones broken GitHub repos, auto-fixes deprecated dependencies, and makes them run again.

## Frontend

Premium marketing landing page built with Vite, React 19, and Tailwind CSS 3.

```bash
cd frontend
npm install
npm run dev
```

## Stack

- React 19 + Vite
- Tailwind CSS 3
- Framer Motion, GSAP, Lenis
- React Three Fiber (hero 3D scene)

# RepoRevive

AI-powered tool that clones broken GitHub repos, auto-fixes deprecated dependencies, and makes them run again.

## Frontend

Premium marketing landing page built with Vite, React 19, and Tailwind CSS 3.

```bash
cd frontend
npm install
npm run dev
```

## Backend

Express + TypeScript API that clones repos into isolated Docker containers, detects the stack, runs them, and (soon) auto-fixes failures with an AI loop. See [backend/README.md](backend/README.md).

```bash
cd backend
npm install
npm run dev
```

## Stack

- React 19 + Vite
- Tailwind CSS 3
- Framer Motion, GSAP, Lenis
- React Three Fiber (hero 3D scene)
